# Dicas do Terminal
vou listar alguns que comando que estamos aprendendo:

- `cd` (navega entre pastas)
    exemplo para entrar numa pasta:

```
$ cd <nome-Pasta>
```    
exemplo de como sair de uma pasta:
```
$ cd ..
```
- `mkdir` (criar pastas)
    exemplo de como criar pastas:
```
$mkdir <nome-da-pasta>

```
